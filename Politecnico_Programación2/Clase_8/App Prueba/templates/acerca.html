Acerca de esta app

Esta aplicación de ejemplo muestra un CRUD simple de cursos usando:

  • Flask 3
  • SQLite (archivo instance/app.db)
  • SQLAlchemy para ORM
  • Flask-Login para autenticación
  • Jinja2 para plantillas
  • Bootstrap 5 para una interfaz agradable
